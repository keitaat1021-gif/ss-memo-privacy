# SS MEMO プライバシーポリシー

最終更新日: 2026年1月20日

## 1. はじめに

SS MEMO（以下「本拡張機能」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めています。本プライバシーポリシーは、本拡張機能がどのようにユーザーの情報を取り扱うかを説明します。

## 2. 収集する情報

本拡張機能は、ユーザーの個人情報を収集しません。

ユーザーがスプレッドシートに入力したデータは、Chrome の同期ストレージ（chrome.storage.sync）を使用してユーザー自身の Chrome アカウントに保存されます。このデータは開発者を含む第三者に送信されることはありません。

## 3. データの使用

保存されたデータは以下の目的でのみ使用されます：
- ユーザーのスプレッドシートデータを保存し、次回使用時に復元する
- ユーザーの複数デバイス間でデータを同期する（Chrome の同期機能を通じて）

## 4. データの共有

本拡張機能は、ユーザーのデータを第三者と共有しません。すべてのデータはユーザーの Chrome アカウント内にのみ保存されます。

## 5. データのセキュリティ

データは Chrome の組み込みストレージ API を使用して安全に保存されます。外部サーバーへのデータ送信は一切行いません。
